LLVM Clang



1. LLVM FrontEnd :

  • lexical-syntax-semantic, output AST

  • AST → CodeGen → IR (.ll)


2. MiddleEnd :

  • .ll → Optimization → .ll


3. Useful commands :

http://clang.llvm.org/get_started.html


clang -E

preprocess only

clang -fsyntax-only

check for correctness, type checking

clang -Xclang -ast-dump -fsyntax-only

dump the AST

clang -cc1 -ast-print

clang -S -Xclang -ast-print file.c

-Pretty printing from the AST to screen

-Output to file.s (not assembly but C or C++ code)

clang -S -emit-llvm file.c

Output unoptimized IR in text format (file.ll)

clang -S -emit-llvm -O3

Optimized IR

clang -S -O3 file.c

Output native machine code (file.s assembly code)

llc -march=cpp -o code.cpp work.ll

.ll to cpp Backend, see llc --version

llc foo.ll

To compile LLVM IR further to assembly, use the llc tool

llvm-as work.ll -o work.bc

llvm-dis work.bc

.ll to .bc (bitcode)

.bc to .ll

llvm-nm work.bc

Name demangling, doesn't work with C++



-cc1 Compiler front-end, and not the driver, should be run

The 'clang' driver is designed to work as closely to GCC as possible to maximize portability


-S Only run preprocess and compilation steps (noLinking)

-c Only run preprocess, compile, and assemble steps

-Xclang <arg> Pass <arg> to the clang compiler

Builtin & Attribute

Attribute: https://en.wikipedia.org/wiki/Attribute_(computing)

Clang Builtin&Attribute Extensions (not available in GCC): http://clang.llvm.org/docs/LanguageExtensions.html


Step by Step to Creat a builtin Function using __attribute__ : __attribute__TOTO(ID)

1. Add definition in : tools/clang/include/clang/Basic/Attr.td

2. Creat a Handler in : tools/clang/lib/Sema/SemaDeclAttr.cpp

3. Add builtin definition for IR : include/llvm/IR/Intrinsics.td

4. CodeGen : tools/clang/lib/CodeGen/CodeGenFunction.cpp


Step by Step to Creat a builtin Function using __builtin__ : X __builtin__TOTO( Y arg1, Z arg2, …)

1. Add definition in : tools/clang/include/clang/Basic/Builtins.def

2. Add builtin definition for IR : include/llvm/IR/Intrinsics.td

3. CodeGen EmitBuiltinExpr : tools/clang/lib/CodeGen/CGBuiltin.cpp
